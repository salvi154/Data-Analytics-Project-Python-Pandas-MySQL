# Data-Analytics-Project-Python-Pandas-MySQL

This project demonstrates an end-to-end data analytics pipeline built using Python, Pandas, and MySQL.
The goal of the project is to extract, clean, transform, and analyze retail order data to answer key business questions using SQL.
